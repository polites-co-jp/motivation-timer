# やる気タイマー — 外部設計書

## 1. 概要

ブラウザ上で動作するポモドーロ式タイマーアプリケーション。
25分の作業セッションと5分の休憩を自動で繰り返し、集中と休息のリズムを作る。
作業開始時にやる気を高める演出（カウントダウンビープ・モチベーションメッセージ）を行う。

## 2. 対象ユーザー

- 集中して作業に取り組みたい人
- 作業と休憩のリズムを自動管理したい人

## 3. 機能一覧

| ID | 機能名 | 説明 |
|----|--------|------|
| F-01 | タイマー | 25分の作業時間をカウントダウン表示する |
| F-02 | 休憩タイマー | 5分の休憩時間をカウントダウン表示する |
| F-03 | 自動遷移 | 作業→休憩→作業を自動で切り替える |
| F-04 | カウントダウン演出 | 作業開始前に「3, 2, 1」のビープ音付きカウントダウンを行う |
| F-05 | やる気メッセージ | 作業開始時にランダムなモチベーションメッセージを表示する |
| F-06 | 円形プログレスバー | 残り時間を円形のプログレスバーで視覚的に表示する |
| F-07 | セッション表示 | 現在何セッション目かをドット等で表示する |
| F-08 | 長い休憩オプション | チェックボックスONの場合、4セッション後に25分の長い休憩を取る |
| F-09 | リセット | タイマーを初期状態に戻し、セッション数もリセットする |
| F-10 | 終了通知 | 作業・休憩終了時に通知音を鳴らす |
| F-11 | グラデーション背景 | 作業中は青系、休憩中は緑系のうねうね動くグラデーション背景を表示する |

## 4. 画面構成

### 4.1 画面一覧

アプリは単一ページ（SPA）で構成され、状態によって表示が切り替わる。

| 状態 | 画面表示 |
|------|----------|
| 待機中 (idle) | スタートボタン、長い休憩チェックボックス |
| カウントダウン中 (countdown) | 「3, 2, 1」のオーバーレイ表示 + ビープ音 |
| メッセージ表示中 (motivated) | やる気メッセージのオーバーレイ（約2秒、フェードアウト） |
| 作業中 (working) | 円形プログレスバー + 残り時間 + リセットボタン |
| 休憩中 (resting) | 円形プログレスバー + 残り時間 + リセットボタン |

### 4.2 画面遷移図

```
[idle]
  │ スタートボタン押下
  ▼
[countdown] ── 3, 2, 1, ビープ（約4秒）
  │
  ▼
[motivated] ── やる気メッセージ表示（約2秒）
  │
  ▼
[working] ── 25:00 からカウントダウン
  │ 残り時間 0:00
  ▼
[resting] ── 5:00 からカウントダウン（長い休憩ON かつ 4セッション後は 25:00）
  │ 残り時間 0:00
  ▼
[countdown] ── 次の作業セッション開始（繰り返し）
```

※ どの状態からでもリセットボタンで `idle` に戻る。

## 5. 操作仕様

### 5.1 スタートボタン

- 待機中 (idle) にのみ表示される
- 押下するとカウントダウン演出が開始される

### 5.2 リセットボタン

- 作業中・休憩中・カウントダウン中・メッセージ表示中に表示される
- 押下すると即座に idle 状態に戻る
- セッション数もリセットされる

### 5.3 長い休憩チェックボックス

- 待機中 (idle) にのみ表示・操作可能
- ON の場合: 4セッション完了後の休憩が25分になる
- OFF の場合: 常に5分休憩

## 6. 演出仕様

### 6.1 カウントダウン演出 (F-04)

- 画面中央にオーバーレイで「3」→「2」→「1」を1秒ずつ表示
- 各数字の表示と同時に短いビープ音を再生
- 「1」の後に長めのビープ音を再生し、作業タイマーを開始
- 休憩終了→次の作業開始時にも同じ演出を行う

### 6.2 やる気メッセージ (F-05)

- カウントダウン演出の直後に画面中央に表示
- あらかじめ用意した複数のメッセージからランダムに1つ選出
- フェードインして約2秒表示後、フェードアウト
- フェードアウト完了後にタイマーのカウントダウンが開始

#### メッセージ例

- 「集中して最高の25分にしよう！」
- 「小さな一歩が大きな成果になる」
- 「今この瞬間に全力を注ごう」
- 「やると決めたあなたはもう半分成功している」
- 「深呼吸して、さあ始めよう」

### 6.3 グラデーション背景 (F-11)

- 作業中: 青系のグラデーションが緩やかにうねうね動く
- 休憩中: 緑系のグラデーションが緩やかにうねうね動く
- 待機中: ニュートラルな色合い
- 状態遷移時にグラデーションの色が滑らかに切り替わる

### 6.4 通知音 (F-10)

- 作業終了時: チャイム音を再生
- 休憩終了時: チャイム音を再生（カウントダウン演出に続く）

## 7. 非機能要件

| 項目 | 内容 |
|------|------|
| 対応ブラウザ | Chrome, Firefox, Edge（モダンブラウザ） |
| レスポンシブ | モバイル・デスクトップ両対応 |
| タブ非アクティブ時 | タイマーが正確に動作すること（経過時間ベースで計算） |
| 外部依存 | 音声はWeb Audio APIで生成し、外部ファイルに依存しない |
